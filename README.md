# 🤖 Face Recognition Attendance System

## 📌 Project Overview

The Face Recognition Attendance System is an AI-powered application that automatically identifies registered users using facial recognition and records attendance without manual intervention.

---

## ✨ Features

- Face Registration
- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Attendance Report Generation
- CSV Attendance Storage

---

## 🛠 Technologies Used

- Python
- OpenCV
- Face Recognition
- NumPy
- Pandas
- Jupyter Notebook

---

## 📂 Project Workflow

1. Capture Face Images
2. Register Users
3. Train Face Recognition Model
4. Detect Faces
5. Recognize Users
6. Mark Attendance
7. Save Attendance in CSV
